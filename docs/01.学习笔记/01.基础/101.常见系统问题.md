## Linux服务器(Java应用)CPU100%，不借助可视化工具怎么定位问题?

1.top 找出进程 CPU 比较高 PID

2.top -Hp PID 打印 该 PID 进程下哪条线程的 CPU 占用比较高 tid

3.printf “%x\\n” tid 将该 id 进行 16 进制转换 tidhex

4.jstack PID \|grep tidhex 打印线程的堆栈信息
